# ไฟฉายแบบตึงๆของนายศธร

แอปพลิเคชันไฟฉายที่สามารถเปิด/ปิดไฟฉาย, ปรับความถี่การกระพริบไฟฉายได้ และสลับระหว่างภาษาไทยและภาษาอังกฤษได้ นอกจากนี้ยังมีหน้าจอ About Me เพื่อแสดงข้อมูลเกี่ยวกับผู้พัฒนา

## คุณสมบัติของแอปพลิเคชัน
- เปิด/ปิดไฟฉาย
- ปรับความถี่การกระพริบไฟฉายได้
- สลับระหว่างภาษาไทยและภาษาอังกฤษ
- ปุ่ม About Me แสดงข้อมูลเกี่ยวกับผู้พัฒนา

## การติดตั้งและการใช้งาน
1. **Clone โปรเจกต์นี้** ลงในเครื่องของคุณ
2. ติดตั้ง dependencies ทั้งหมด
3. รันแอปบนอุปกรณ์หรือ Emulator เพื่อทดสอบ

## การ build ไฟล์ APK
หากต้องการสร้างไฟล์ APK เพื่อการติดตั้งบนอุปกรณ์ Android จริง คุณสามารถสร้างไฟล์ได้ในโหมด release

## การทดสอบ
คุณสามารถทดสอบแอปและการทำงานต่างๆ ได้ ทั้งในส่วนของ unit test และ integration test เพื่อให้มั่นใจว่าแอปทำงานได้ตามที่คาดหวัง

## ข้อกำหนด
- **Flutter SDK**: เวอร์ชัน 2.0 ขึ้นไป
- **Dart SDK**: เวอร์ชัน 3.0 ขึ้นไป
